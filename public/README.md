# Public Directory

Place your static assets here:

## Required Images

-   `favicon.ico` - Site favicon (16x16 or 32x32)
-   `apple-touch-icon.png` - Apple touch icon (180x180)
-   `icon-192.png` - PWA icon (192x192)
-   `icon-512.png` - PWA icon (512x512)
-   `logo.png` - Your business logo
-   `og-image.jpg` - Open Graph image for social sharing (1200x630 recommended)

## Optional

-   Any other static assets (images, PDFs, etc.)

## Image Optimization Tips

-   Use WebP or AVIF format when possible
-   Compress images before uploading
-   Keep file sizes small for faster loading
-   Use descriptive filenames for better SEO
