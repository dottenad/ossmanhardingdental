Brandon Grotesque Font Files Required
=====================================

Brandon Grotesque is a commercial font. You need to purchase and add the following font files to this directory:

Required files (in WOFF2 format for best web performance):
- BrandonGrotesque-Regular.woff2
- BrandonGrotesque-Medium.woff2
- BrandonGrotesque-Bold.woff2

You can purchase Brandon Grotesque from:
- MyFonts: https://www.myfonts.com/fonts/hvdfonts/brandon-grotesque/
- Adobe Fonts (if you have Creative Cloud): https://fonts.adobe.com/fonts/brandon-grotesque

If you have the font in OTF or TTF format, you can convert them to WOFF2 using:
- https://cloudconvert.com/ttf-to-woff2
- https://transfonter.org/

Alternative: If you don't have Brandon Grotesque, you can use a similar Google Font:
- Update lib/config.ts: change headingFont to "Raleway" or "Montserrat"
